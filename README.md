## 🚀 Chức năng

### 📶 1. Tính toán dữ liệu 4G
- Tính dung lượng theo giờ
- Tính dung lượng theo ngày
- Tính dung lượng theo tháng
- Đề xuất gói SIM phù hợp

### 📷 2. Tra cứu thông tin Camera
- Hình ảnh từng model
- Độ phân giải
- Chuẩn nén H.265
- Bitrate tối đa
- Pin / Nguồn
- AI
- Kiểu lắp đặt

### 💾 3. Tính dung lượng HDD *(Đang phát triển)*

### 🔍 4. Tra cứu Camera *(Đang phát triển)*
